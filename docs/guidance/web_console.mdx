## 使用方法

<Tabs groupId="type">

<TabItem value="lite" label="Serein.Lite" >

在 设置>网页 勾选**启用**

</TabItem>
<TabItem value="plus" label="Serein.Plus">

在 设置>网页 勾选**启用**

</TabItem>

<TabItem value="cli" label="Serein.Cli" >

在控制台输入

```batch
webserver start
```

</TabItem>

</Tabs>

默认情况下网页服务器将会在 http://127.0.0.1:50000 上运行，如果一切顺利的话点击该连接就能打开网页控制台了
