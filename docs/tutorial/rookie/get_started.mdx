# 快速上手

:::tip 推荐阅读

[如何选择Serein类型](choose)

:::

## 0. 提示

如果你将要使用的是[`Lite`](choose/#lite)/[`Plus`](choose/#plus)类型的Serein，你可以跳至[2. 下载](#2-下载serein)

`Lite`和`Plus`在缺少运行库时会有弹窗提示，按其提示链接下载即可

## 1. 安装运行库

[![dotnet](https://img.shields.io/badge/8.0-512BD4?style=for-the-badge&logo=.net)](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

:::note

>[.NET 8.0 支持的系统](https://github.com/dotnet/core/blob/main/release-notes/8.0/supported-os.md)

.NET 8.0 不支持 Windows 8 及以下的版本（理论上[^1]），但是你可以在这些系统上使用[旧版本的Serein](/docs/1.x/tutorial/install/)

:::

| Serein类型 | 需要的运行时类型 |
| :--------: | :--------------: |
|    Cli     |   .NET 运行时    |
|    Lite    | .NET 桌面运行时  |
|    Plus    | .NET 桌面运行时  |

:::tip
.NET 桌面运行时 包含 .NET 运行时，因此你可以选择直接下载前者
:::

## 2. 下载Serein

### 发行版

即稳定版，有着较少的Bug

[![GitHub Release](https://img.shields.io/github/v/release/SereinDev/Serein?style=for-the-badge)](https://github.com/SereinDev/Serein/releases/latest)

### 测试版

包含一些前瞻性功能或Bug修复

:::warning

可能存在未知的漏洞，与发行版相比更不稳定，在使用测试版前务必**备份数据**

:::

- [![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/SereinDev/Serein/.github%2Fworkflows%2Fbuild.yml?style=for-the-badge)](https://github.com/SereinDev/Serein/actions/workflows/build.yml)  
- [nightly.link](https://nightly.link/SereinDev/Serein/workflows/build/main?preview)

### 自行编译

详见[自行编译](../advanced/build)

## 3. 运行

<Tabs>
<TabItem value="windows" label="Windows" >

双击运行即可

</TabItem>
<TabItem value="other" label="Linux / macOS">

解压后在终端输入

```sh
cd path/to/serein # 替换为你的目录
sudo chmod +x ./Serein.Cli # 赋予执行权限

./Serein.Cli
```

</TabItem>
</Tabs>

[^1]: 你可以自行尝试[此教程](https://blog.csdn.net/qq_29998377/article/details/144239719)中提供的方法
