# 输入或输出乱码

## 输出

如果你启动的是Java服 或 批处理文件 或 Python程序，或是控制台中出现了黑色小方块（见下），你可以尝试在 **服务器配置>输入/输出** 中修改输出编码为 **`GBK`** 或其他值

```txt

```

:::note

修改输出编码需要重启服务器进程才能生效

:::

## 输入

:::warning

基岩版服务器（BDS）的输入乱码问题貌似无一劳永逸的解法

如果你输入只有英文指令 或 只有`tellraw`而不涉及其他指令的话，你可以尝试在 **服务器配置>输入/输出** 中开启 **使用Unicode字符**

:::
