## 查询参数

| 参数名  | 说明     | 必需                  |
| ------- | -------- | --------------------- |
| `token` | 鉴权凭证 | 否，[见下](#鉴权凭证) |

### 鉴权凭证

当 设置 > 网页 > 鉴权凭证 不为空时，你需要提供一个凭证

```uri
/ws/connection?token=your_token
```

未提供凭证或凭证不正确时连接会被立即断开

## 数据结构

所有广播消息均使用 JSON 文本传输

```json
{
    "type": "information",
    "origin": "Serein",
    "data": "Hello World."
}
```

| 字段名   | 说明     | 类型               |
| -------- | -------- | ------------------ |
| `type`   | 广播类型 | `string`           |
| `origin` | 数据来源 | `string`           |
| `data`   | 数据主体 | `string` \| `null` |
