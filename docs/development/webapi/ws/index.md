# WebSocket 广播

用于广播服务器状态变化、输入输出和日志消息

```txt
/broadcast?id=survival1&token=your_token
```

:::note

客户端发送的一切消息都会被忽略

:::

## 查询参数

| 参数名  | 说明     | 必需     |
| ------- | -------- | -------- |
| `id`    | 服务器 Id | 是       |
| `token` | 鉴权凭证 | 否，[见下](#鉴权凭证) |

### 服务器 Id

要监听服务器的 Id，若指定的服务器不存在则连接会被立即断开

### 鉴权凭证

当 设置>WebApi>鉴权凭证 不为空时，你需要提供一个凭证

未提供凭证或凭证不正确时连接会被立即断开
