# 启动服务器

```http
POST /servers/{id}/start HTTP/1.1
```

## 请求

### 方法

`POST`

### 路径参数

| 参数名 | 说明      |
| ------ | --------- |
| `id`   | 服务器 Id |

## 响应

### 状态码

- `200`
  - 启动成功（当未使用虚拟终端时）
- `202`
  - 启动成功（当使用虚拟终端时）
- `403`
  - 服务器已在运行
  - 启动文件为空
- `404`
  - 未找到指定的服务器
