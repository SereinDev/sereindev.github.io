# 重新启动服务器

```http
POST /servers/{id}/restart HTTP/1.1
```

## 请求

### 方法

`POST`

### 路径参数

| 参数名 | 说明      |
| ------ | --------- |
| `id`   | 服务器 Id |

## 响应

### 状态码

- `202`
  - 请求重启成功
- `403`
  - 服务器未在运行
- `404`
  - 未找到指定的服务器

### 响应示例

```json
{
  "data": null,
  "errorMsg": null,
  "details": [],
  "time": "2024-10-12T20:43:40.4236248+08:00"
