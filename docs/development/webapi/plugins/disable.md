# 禁用指定插件

```http
POST /plugins/{id}/disable HTTP/1.1
```

## 请求

### 方法

`POST`

### 路径参数

| 参数名 | 说明    |
| ------ | ------- |
| `id`   | 插件 Id |

## 响应

### 状态码

- `202`
  - 成功禁用
- `403`
  - 插件已被禁用
- `404`
  - 插件不存在

### 响应示例

```json
{
  "data": null,
  "errorMsg": null,
  "details": [],
  "time": "2025-07-04T19:53:26.8462963+08:00"
}
```
