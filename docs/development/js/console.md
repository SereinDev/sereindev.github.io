# 控制台

## 输出信息

```ts
console.log(...data: any[]): void
serein.console.log(...data: any[]): void

console.info(...data: any[]): void
serein.console.info(...data: any[]): void
```

| 参数   | 类型    | 说明       |
| ------ | ------- | ---------- |
| `data` | `any[]` | 输出的内容 |
