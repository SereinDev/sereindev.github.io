# 储存

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

为插件提供便捷的字符串储存功能

## 类型声明

<Tabs groupId="lang">

<TabItem value="cs" label="C#" >

```cs
namespace Serein.Core.Services.Plugins.Storages;

public partial class StorageBase

    public int Length { get; }
```

## 本地储存

保存在本地文件（`Serein/plugins/local-storage.json`）中

:::tip

你可以借助此功能实现一些简单的配置文件的储存

:::

<Tabs groupId="lang">

<TabItem value="cs" label="C#" >

```cs title="示例"
var localStorage = serviceProvider.GetRequiredService<LocalStorage>();
localStorage.SetItem("key", "value");
```

</TabItem>

<TabItem value="js" label="JavaScript" >

```ts
declare var localStorage: Storage;
```

与浏览器中的[`Window.localStorage`](https://developer.mozilla.org/zh-CN/docs/Web/API/Window/localStorage)一致

</TabItem>
</Tabs>

## 会话储存

仅保存在内存中，退出后数据丢失

:::tip

通常用于储存在同一个Serein生命周期内可以重复利用的数据

:::

<Tabs groupId="lang">

<TabItem value="cs" label="C#" >

```cs title="示例"
var sessionStorage = serviceProvider.GetRequiredService<SessionStorage>();
sessionStorage["key"] = "value";
```

</TabItem>

<TabItem value="js" label="JavaScript" >

```ts
declare var sessionStorage: Storage;
```

与浏览器中的[`Window.sessionStorage`](https://developer.mozilla.org/zh-CN/docs/Web/API/Window/sessionStorage)一致

</TabItem>
</Tabs>
